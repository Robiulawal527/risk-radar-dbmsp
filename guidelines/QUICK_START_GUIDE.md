## 🔧 Main Features Guide

### 1. Viewing Crime Data

1. **Navigate to Dashboard**
   - Click/tap on "Dashboard" in navigation menu

2. **Use Filters**
   - Crime Type: Select specific crime categories
   - Date Range: Filter by time period
   - Area: Focus on specific neighborhoods
   - Click "Apply Filters" to update map

3. **Interact with Map**
   - Zoom in/out using mouse wheel or pinch gesture
   - Click on crime markers for details
   - Heatmap shows crime density (red = high risk)

### 2. Reporting a Crime

1. **Click "Report Crime" button**
2. **Fill in Details:**
   - Crime type
   - Location (or use current location)
   - Description
   - Upload photos (optional)
3. **Submit Report**
4. **Track status in Reports section**

### 3. Emergency Features

**SOS Button:**
- Instantly alerts nearby police
- Sends your GPS coordinates
- Activates emergency contacts

**Feeling Unsafe:**
- Share live location with trusted contacts
- Get nearest police station info
- Access emergency helplines

### 4. Finding Safe Routes

1. **Navigate to "Safe Route" section**
2. **Enter:**
   - Starting location
   - Destination
3. **View Options:**
   - Safest route (highlighted in green)
   - Alternative routes with risk scores
   - Estimated travel time
4. **Save frequently used routes**

### 5. Analytics & Insights

**Crime Analytics:**
- Trends over time (daily, weekly, monthly)
- Crime type distribution
- High-risk area identification
- Peak crime hours

**Predictions (AI-powered):**
- Forecast crime hotspots
- Risk trend analysis
- Area safety scores

## ❓ Troubleshooting

### Login Issues
1. **"Invalid email or password"**
   - Verify you're using correct credentials
   - Check caps lock is off
   - Try copying credentials from this guide

2. **Cannot Access After Login**
   - Clear browser cache
   - Try incognito/private mode
   - Refresh the page

### Map Not Loading
1. Check internet connection
2. Enable location permissions
3. Refresh the page
4. Try a different browser

### Mobile App Issues
1. **App crashes on startup:**
   - Clear app cache
   - Reinstall app
   - Check for updates

2. **Location not working:**
   - Enable GPS/location services
   - Grant location permission in app settings
   - Restart the app

## 📊 Understanding Risk Levels

**Low Risk (Green):**
- 0-2 crimes per month
- Safe for all times
- Residential areas, parks

**Medium Risk (Yellow):**
- 3-5 crimes per month
- Exercise caution at night
- Busy commercial areas

**High Risk (Orange):**
- 6-10 crimes per month
- Avoid if possible at night
- Crowded markets, stations

**Critical Risk (Red):**
- 10+ crimes per month
- High alert required
- Avoid especially after dark
